Specs & usage

Display height
44–56 px (signature) · keep aspect ratio
Export resolution
2× or 3× PNG for retina (e.g. 120 px tall source)
Clear space
≥ half the hex height on all sides
File format
PNG (email) · SVG (web) — both delivered
Do not
recolor · stretch · add shadows · place on busy photos

Tip: in Gmail / Outlook, host the PNG at a public URL and reference it with an <img> tag inside the signature, sizing it with height="48". PNG files are in the signature-assets/ folder — 96px (2×)  144px (3×), on white and reversed-on-navy. Ask me for any other size.
